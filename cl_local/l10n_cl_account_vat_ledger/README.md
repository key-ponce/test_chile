# l10n_cl_account_vat_ledger
Chilean Account VAT Ledger

##Chilean VAT Sales and Purchases Ledger
========================================

Module that provides a Sales and Purchases ledger, in order to declare VAT tax. It allows to export the ledger in xls format.

## Installation
 
### Dependencies
aeroo_reports

(we advise to use in conjunction with a Docker aeroo-docs implementation)

##Libro de compras y ventas para localización Chilena
=====================================================

## Descripción

Módulo que proporciona un informe (libro) de compras y ventas, con la finalidad de declarar el IVA. Permite exportar a formato xls dicho informe.

## Instalación
 
### Dependencias
aeroo_reports

(recomendamos utilizar en conjunto con una implmentación de aeroo-docs bajo Docker)
