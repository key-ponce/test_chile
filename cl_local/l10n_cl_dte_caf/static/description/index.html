Chile - Administración de Códigos de autorización de folios para DTEs

Permite administrar y asignar los códigos de autorización de folios para los Documentos Tributarios Electrónicos (DTE)

* Integración total con el resto de los sistemas de Localización Chilena.

* Permite ingresar a Odoo, y administrar estado, asignación de DTEs a distintas secuencias de Documentos Tributarios.

    Valida:

  • Si el RUT del archivo de CAF, se corresponde con el RUT de la compañía
  • Si la secuencia está fuera de rango.
  • Si la secuencia se corresponde con el tipo de documento autorizado por el CAF.

    • * Muestra el porcentaje de utilización del archivo, advirtiendo al usuario si debe renovar el CAF para continuar la facturación.
